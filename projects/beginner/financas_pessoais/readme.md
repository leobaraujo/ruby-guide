# Finanças Pessoais

Projeto desenvolvido em Ruby puro, executado via linha de comando (CLI), para auxiliar no controle de finanças pessoais. A aplicação permite cadastrar usuários, categorias e transações (saques e depósitos), utilizando o SQLite3 como banco de dados.

## Funcionalidades

- Cadastro de Usuários
- Cadastro de Categorias
- Cadastro de Transações

## Rubygems necessárias:
- terminal-table 4.0.0
- sqlite3 2.6.0

> Para iniciar, execute `./bin/start.rb` no terminal.
